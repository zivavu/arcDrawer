# arcDrawer

How to use?
You kinda have to play with the arcs values to understand them but bassicly:

**Arc** - single line that is drawn on mouse move

Line Witdh - starting Arc width

Number of Lines - number of segments that every Arc is made of

Line Decay - amount of width decay with every another segment of single Arc

Arc Stability - multiplier of the Arc direction sustain(segments become longer and more stable towards Arc's random direction)

Arc Unstability - amount of segments random direction(segments become longer but they are wilder as well)

Hue Variation - amount of color hue randomness of every Arc

Saturation - amount of arc saturation(not random)

Shortcuts:

Ctrl + Z to undo last change

Ctrl + C to change the arc color

Ctrl + R to clear the drawing area

Arrow keys are used to set the direction of shadow when it is enabled but that function turned out to be kinda pointless
